Overview

This repository provides the datasets, MATLAB codes, and figures used in the paper. The multiscale model couples whole-body blood circulation with cerebral autoregulation and network pathway analysis to investigate hemodynamic redistribution under anatomical variations and stenosis conditions.

Abstract

Cerebral hemodynamics are fundamentally regulated by the Circle of Willis (CoW), which redistributes blood flow via communicating arteries to maintain cerebral perfusion under anatomical variations and vascular stenosis. In this study, we develop a multiscale circulation model by coupling a systemic hemodynamic framework with an image-based cerebral arterial network reconstructed from medical imaging data.

The model integrates a cerebral autoregulation mechanism (CAM) and enables quantitative analysis of flow redistribution across the CoW under normal, anatomically varied, and pathological stenosis conditions. Baseline simulations reproduce physiological flow distributions with negligible collateral activation, consistent with clinical observations.

In contrast, anatomical variations and progressive stenosis induce distinct collateral pathway recruitment patterns, with early activation of the anterior communicating artery (ACoA) and structure-dependent engagement of posterior communicating arteries (PCoAs). A pathway-based source-to-sink flow analysis is further introduced to quantify the dynamic reconfiguration of collateral pathways in response to structural changes in the vascular network.

The proposed framework provides a physiologically interpretable and image-informed tool for investigating cerebral blood flow regulation through functional collaterals, with potential applications in the diagnosis, prognosis, and treatment planning of cerebrovascular diseases.

Data

All data are anonymized and aggregated. Time series are provided in MATLAB (.mat) and CSV (.csv) formats. Variable names and units are described in the README files below.

File Type Size Description
hemo_timeseries.mat MAT ~45 MB Simulated pressure and flow time series for major arteries, veins and cerebral vessels across all scenarios.
inputs_parameters.csv CSV ~200 kB Model parameters and boundary conditions (vessel radii, lengths, compliances, stenosis degrees, heart parameters, etc.).
cam_outputs.csv CSV ~300 kB Summary outputs of cerebral autoregulation (flow redistribution indices, perfusion metrics in territories, etc.).

MATLAB code

The core MATLAB scripts for running the multiscale circulation–CAM model and reproducing the main figures.

File Type Size Description
run_main.m M ~15 kB Main entry script to run the full simulation pipeline.
config_parameters.m M ~8 kB Set model parameters, stenosis conditions and simulation scenarios.
plot_figures.m M ~12 kB Generate the main figures used in the paper.
README.txt TXT ~4 kB Instructions on how to set up the environment and run the code.

Reproducibility

  1. Download this repository or clone it from GitHub.
  2. Open MATLAB (R20XXa or later is recommended).
  3. Add the code folder to the MATLAB path.
  4. Edit config_parameters.m if you wish to change scenarios.
  5. Run run_main.m to generate the simulation outputs.
  6. Run plot_figures.m to reproduce the main figures.
